fase atual:

1º Fase - Análise Curricular

  • Critérios: Compatibilidade com o perfil da vaga e relevância das experiências apresentadas.

Próximas fases:

2º Fase - Regra de Negócio e Prototipação

  • Objetivo: Desenvolver um protótipo em Figma baseado em uma regra de negócio fictícia.
  • Critérios: Criatividade, funcionalidade e clareza no protótipo.
  • Entrega: Um link com o protótipo no Figma. para o e-mail: psi@educargames.com com o assunto "nome-do-candidato 2º Fase"
  • Data: 28/11/2024 até ás 18 horas.

3º Fase - Estrutura HTML, CSS, Organização no GitHub e JavaScript

  • Objetivo: Construir uma página estática aplicando boas práticas de HTML e CSS, com publicação no GitHub e adicionar funcionalidades interativas ao protótipo usando JavaScript.
  • Critérios: Semântica, responsividade, organização do repositório no GitHub e qualidade do código, usabilidade e clareza das interações implementadas.
  • Entrega: Link do repositório compartilhado no GitHub para o e-mail: psi@educargames.com com o assunto "nome-do-candidato 3º Fase".
  • Data: 09/12/2024

4º Fase - Apresentação do Pitch e Deploy da Página

  • Objetivo: Apresentar o projeto e realizar o deploy da página.
  • Critérios: Clareza na apresentação, funcionalidade no ambiente publicado e explicação das escolhas técnicas.
  • Entrega: Link para o deploy do projeto e apresentação em até 10 minutos para o e-mail: psi@educargames.com com o assunto "nome-do-candidato 4º Fase".
  • Data: será anunciado após a entrega da 3ª Fase. (Fique atento! O anúncio será feito por e-mail)

5º Fase - Resultado Final

  • Objetivo: Avaliar o desempenho geral ao longo do processo.
  • Critérios: Desempenho técnico, aderência ao perfil desejado e potencial de desenvolvimento.
  • Entrega: Será realizada pela equipe avaliadora.
  • Data: Dois dias úteis após a entrega da 4º Fase.
